1. Explique o que é herança em orientação a objetos e como ela é implementada em C#. Dê um exemplo prático de como a herança pode ser utilizada em um sistema de gerenciamento de funcionários.

Resposta: 
A parte de herança em OO é bem importante, com ela podemos economizar linhas de códigos e fazer códigos mais limpos, ela é implementada em C# por uma Classe-Pai, onde ficará os métodos e atributos base das Classes-filho, para utilizá-la utilizamos os dois pontos ":", antes dele será colocado o nome da classe filho,  um exemplo: "Admissao : Funcionarios". Utilizando a herança na classe Admissao(Classe-Filha), é possivel adicionar novos atributos, além de herdar os atributos e métodos da Classe Funcionarios(Classe-Pai).
